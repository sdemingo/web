
# Este blog no tiene comentarios

Este blog, a diferencia de los clásicos, no tiene posibilidad de escribir comentarios o de interactuar con el texto en la propia página del artículo. Esto es así principalmente por una razón de sencillez.

Todo este blog se construye de forma estática (esto quiere decir usando únicamente HTML y CSS) a través de [un pequeño script artesanal de Python](https://panicerror.org/blog/sobre-este-blog.md.html). No estoy interesado en usar ninguna otra tecnología más allá de estas y por tanto, para evitar usar soluciones complejas he decidido prescindir de los comentarios en los artículos.

Si deseas contactar conmigo para hacerme llegar algún comentario o retroalimentación sobre lo que he escrito puedes usar mi correo electrónico que se muestra en la página inicial.
